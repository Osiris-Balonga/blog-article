# Blog Article

Page d'article de blog structurée avec HTML5 sémantique et images locales pour présenter un article, une sidebar, un tableau de données et une section de commentaires.

## Objectif

Ce projet répond au livrable de l'article de blog du module HTML de la Phase 2. La page met en avant la structure HTML, les paragraphes, les citations, le tableau et le formulaire de commentaire.

## Fonctionnalités

- Header avec titre, auteur, date et catégorie
- Corps de l'article avec paragraphes et bloc de citation
- Figure image avec légende
- Tableau de données intégré
- Sidebar avec articles récents
- Section commentaires et formulaire labellisé

## Technologies

- HTML5 sémantique
- Balises `article`, `aside`, `nav` et `figure`
- Images avec `alt`
- Formulaire accessible

## Structure

```text
blog-article/
  images/
  article.html
  README.md
```
